# PNG-JPG-Ebook-Viewer
An app that lets you view images as if they were a digital book, without having a different app for each book.

Since I'm so tired of all the different apps for every digital book I have (for school in particular), I decided to create my own app, which is definitely better than the original ones.

# HOW IT WORKS

It's easy, just download the [.exe file from the release](https://github.com/Xiarol1299/PNG-JPG-Ebook-Viewer/releases) and run it, and you're done 
(obviously for the .py file, you'll also need to download the necessary libraries; they're listed at the bottom of the requirements).

# What you can do:
1) Write and underline WITH YOUR PEN (I've created a highly optimized system that works great and saves what you write in the original image).

2) View images as if they were a digital book.

3) It save the page when you exit from the app or from one book and, when you re-enter in the same book, the app re-open in the same page

4) Dual page visualization

5) Dark mode(default) / Light mode

# Requirements (only for the .py file)

1) Python
2) Library: PyQt6 (pip install PyQt6)
3) Library: pdf2image (pip install pdf2image)
4) pdf2image requirement: Poppler (add the bin folder in the system variables)
